设计原则

降低认知成本

每个组件服务于"快速决策"——用户不需要理解系统，只需要看到结果。评分颜色、信息层级、CTA 位置都为此优化。

Mobile First 决策加速

移动端用户在做决策，不是浏览内容。Top Match 优先展示，Sticky CTA 始终可见，信息密度压缩到决策关键维度。

SEO = 决策工具

每个 SEO 页面都是完整的决策工具：AI 摘要 → 评分排序 → 对比 → FAQ。不是内容堆砌，是结构化的决策指南。
